<h2><a href="https://leetcode.com/problems/odd-even-linked-list/">328. Odd Even Linked List</a></h2>
<h3>Medium</h3>
<hr>
<p>Given the <strong>head</strong> of a singly linked list, group all the nodes with odd indices together followed by the nodes with even indices, and return <em>the reordered list</em>.</p>

<p>The first node is considered odd, the second node even, and so on.</p>

<p>You must do this in <strong>O(1)</strong> extra space and <strong>O(n)</strong> time.</p>

<h3>Example 1:</h3>
<pre>
<b>Input:</b> head = [1,2,3,4,5]
<b>Output:</b> [1,3,5,2,4]
</pre>

<h3>Example 2:</h3>
<pre>
<b>Input:</b> head = [2,1,3,5,6,4,7]
<b>Output:</b> [2,3,6,7,1,5,4]
</pre>

<hr>
<h3>Constraints:</h3>
<ul>
  <li><code>n ==</code> number of nodes in the list</li>
  <li><code>0 &lt;= n &lt;= 10<sup>4</sup></code></li>
  <li><code>-10<sup>6</sup> &lt;= Node.val &lt;= 10<sup>6</sup></code></li>
</ul>
