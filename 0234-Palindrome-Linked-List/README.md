<h2><a href="https://leetcode.com/problems/palindrome-linked-list/">234. Palindrome Linked List</a></h2>
<h3>Easy</h3>
<hr>
<p>Given the <strong>head</strong> of a singly linked list, return <code>true</code> if it is a palindrome, or <code>false</code> otherwise.</p>

<h3>Example 1:</h3>
<pre>
<b>Input:</b> head = [1,2,2,1]
<b>Output:</b> true
</pre>

<h3>Example 2:</h3>
<pre>
<b>Input:</b> head = [1,2]
<b>Output:</b> false
</pre>

<hr>
<h3>Constraints:</h3>
<ul>
  <li>The number of nodes is in the range <code>[1, 10<sup>5</sup>]</code>.</li>
  <li><code>0 &lt;= Node.val &lt;= 9</code></li>
</ul>

<hr>
<h3>Follow-up:</h3>
<p>Could you do it in <strong>O(n)</strong> time and <strong>O(1)</strong> extra space?</p>
